# Pricing

## Free (BSL 1.1)

Everything you need for development, testing, and personal projects.

- Static analysis (130+ rules, 18 languages)
- Semantic analysis (tree-sitter AST, taint, CFG, null safety)
- Software Composition Analysis (10 lockfile formats, Google OSV)
- SARIF, JSON, HTML output
- MCP server mode
- CWE + NIST SP 800-53 compliance mappings
- CI/CD integration (GitHub Actions, GitLab CI)
- Pre-commit hooks
- Diff-only scanning

**Free for:** development, testing, personal projects, education, open-source CI/CD, evaluation.

## Commercial license required

Production use in a commercial product or service requires a commercial license.

**Examples that need a license:**

- Embedding Dojigiri in a SaaS platform
- Running in a commercial CI/CD pipeline
- Bundling in a product you sell
- Offering as a managed service

**Contact:** licensing@dojigiri.com

Volume discounts available for multi-team deployments. White-label licensing available.

## LLM deep scan costs

The LLM tier (Tier 3) sends code to an external API. You pay the LLM provider directly -- Dojigiri adds no markup.

| Project size | Estimated cost per scan |
|-------------|------------------------|
| Small (< 50 files) | $0.05 - $0.20 |
| Medium (50-200 files) | $0.20 - $1.00 |
| Large (200+ files) | $1.00 - $5.00 |

## Open source timeline

Dojigiri uses the Business Source License 1.1 (BSL 1.1). On **2030-03-09**, the current version converts to **Apache License 2.0**, making it fully permissive.

Every version follows the same 4-year pattern: BSL 1.1 at release, Apache 2.0 four years later.
